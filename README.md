# offline-site.sh

> Download any website and all required resources for offline viewing

## Usage

With `wget` installed, execute in your terminal:

```bash
git clone https://github.com/ascribe/offline-site.git
cd offline-site/

./offline-site.sh www.yourwebsite.com
```
